## Default variables
```yaml
aws_iam_role:
  name: "example"
  aws_profile: "{{ _aws_profile }}"
  # Pass either names or ARNs for the role.
  managed_policies: []
  # Which document policy to apply.
  # Only EC2 is implemented for now.
  policy_document: ec2
  tags:
    #Type: "util"
  state: present
  _result: {}

```
